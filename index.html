Quiz

Tous les matins, vous attendez le bus pour vous rendre à l’école ou au boulot. Et tous les matins, en observant les personnes qui vous entourent, vous constatez toujours la même chose : elles tiennent toutes leur sac ou sacoche d’une main, et leur smartphone Android de l’autre. Elles ne quittent jamais leur écran des yeux. Les applications mobiles font partie de notre quotidien ! Vous vous dites : "c’est génial, si je savais développer des applications mobiles Android, toutes ces personnes pourraient les utiliser !” . Vous vous dites également : “ouais mais en fait, c’est hyper compliqué, j’y arriverai jamais”. Eh bien pas du tout, il suffit de commencer par le commencement ! Dans ce cours: vous apprendrez pas à pas les fondamentaux de la programmation Android. Vous découvrirez comment afficher des informations sur l’écran et interagir avec l’utilisateur. Vous verrez comment implémenter une logique de jeu, afin de développer une application ludique et intéressante. Vous êtes prêts à conquérir le monde Android avec votre première application ? Alors allons-y

Contenu du cours

Partie 1 - Créez votre première application

  1. Installez Android studio
  2. Créer le projet et prenez en main l'environnement
  3. Dessiner l'interface utilisateur de votre premiére activité
  4. Référencez les éléments graphiques dans votre activité
  5. Gérez les actions de l'utilisateur
  6. Lancez l'application sur l'emulateur ou sur un équipement réel

Partie 2 - Etendez les fonctionnalités de votre application

  1. Créez une seconde activité
  2. Découvrez l'architecture Modéle-vue_Controleur
  3. Définissez votre premier modéle
  4. Implémentez la logique de jeu dans le controleur
  5. Présentez le score au joueur

Partie 3 - Perfectionnez votre application

  1. Récupérez le score du joueur dans l'activité principale
  2. Mémorisez les préférences du joueur
  3. Améliorez l'ecpérience utilisateur
  4. Comprenez le cycle de vie d'une activité
  5. Tournez votre équipement : que se passe t'il ?
